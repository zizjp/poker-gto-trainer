# 🃏 Poker GTO Trainer

**バージョン**: v0.8.4  
**最終更新**: 2025年11月22日

GTO混合戦略に基づくポーカー学習アプリケーション - プリフロップシナリオをスワイプ操作でマスターしよう！

---

## 🎯 プロジェクト概要

GTO Wizardのデータを活用した、プリフロップ・3Bet・4Betシナリオの学習アプリです。

### 🌟 主な機能

#### ✅ 実装済み（v0.8.4）

1. **カスタムレンジエディター**
   - 169個のハンド全てに対してアクション頻度を設定
   - 頻度検証（合計100%になるまで保存できない）
   - LocalStorageによる自動保存

2. **学習モード（スワイプ式）**
   - 50問のスワイプ学習（重複なし）
   - **Framer Motion**による滑らかなアニメーション
   - ドラッグ中のヒント表示
   - PC対応（マウスドラッグ＋キーボード矢印キー）
   - iPhone対応（タッチスワイプ）

3. **判定モード**
   - **確率的判定**: 混合戦略確率に基づく判定
   - **頻度表示判定**: 50問全体で期待値との誤差を評価

4. **フィードバック機能**
   - 正解/不正解の視覚的表示
   - **解説表示**（4パターンのメッセージ）
   - 1.5秒自動進行

5. **統計・復習機能**
   - 学習履歴の表示
   - **苦手問題復習機能**（正解率50%以下のハンドを抽出）
   - アクション別正解率

6. **UI/UX**
   - **モダンなデザイン**（Emerald/Slateカラーパレット）
   - レスポンシブデザイン（PC/iPhone対応）
   - スクロール無効化（ゲーム中の誤操作防止）
   - 操作ガイド固定表示

## 📱 使い方

### 1. レンジエディター
1. 「レンジを追加」でカスタムレンジを作成
2. 169個のハンドに対してアクション頻度を設定
3. 合計が100%になるまで保存不可（厳密な検証）

### 2. 学習モード
1. **学習設定画面**でレンジ、出題範囲、判定モードを選択
2. **スワイプ操作**：
   - ← 左: Call
   - ↑ 上: Fold
   - → 右: Raise
3. PC: マウスドラッグまたはキーボード矢印キー
4. フィードバック画面で解説を確認
5. 50問完了後、統計画面で結果を確認

### 3. 苦手問題復習
1. 統計画面から「苦手なハンドを復習」をクリック
2. 正解率50%以下のハンドのみで学習セッション開始
